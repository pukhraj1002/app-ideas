# Leaf Condition Detector 

**Tier:** 1-Beginner

The **Leaf Condition Detector** is an AI-powered application designed to help users assess the health of leaves by classifying their condition through machine learning. This tool uses a model trained on various leaf images via Teachable Machine and TensorFlow.js, allowing users to upload an image or use their webcam to analyze a leafâ€™s condition in real time. The model predicts the condition with associated probabilities, offering a detailed insight into potential leaf diseases or health issues. With a responsive user interface and seamless image processing, the application aims to provide an intuitive solution for farmers, gardeners, or researchers to monitor plant health effortlessly. It leverages modern web technologies like HTML, CSS, and JavaScript, making it easily accessible through any web browser.

## User Stories

- [ ] User can upload an image of a leaf using the `upload` button.
- [ ] User can classify leaf conditions using a Teachable Machine model.
- [ ] User can view prediction results along with probabilities for each condition.

## Bonus Features

- [ ] User can classify the condition of a leaf directly from the webcam.
- [ ] User can receive prediction results for uploaded images.
- [ ] Provide detailed accuracy scores for the predictions.
- [ ] Include additional leaf health metrics or explanations based on the predictions.
